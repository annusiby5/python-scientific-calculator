# python-scientific-calculator

A feature rich Scientific Calculator built with Python and Tkinter. The application provides basic arithmetic operations, scientific functions, memory management, calculation history, and a user friendly graphical interface.

## Features

### Basic Operations

* Addition
* Subtraction
* Multiplication
* Division
* Modulus
* Exponentiation

### Scientific Functions

* Square Root
* Sine
* Cosine
* Tangent
* Logarithm (Base 10)
* Factorial

### Memory Functions

* M+ (Memory Add)
* M- (Memory Subtract)
* MR (Memory Recall)
* MC (Memory Clear)

### Additional Features

* Expression Evaluation
* Calculation History
* Timestamped Records
* Save History to File
* Graphical User Interface (GUI)
* Error Handling

## Technologies Used

* Python 3
* Tkinter
* Math Module
* Datetime Module

## Installation

Clone the repository:

```bash
git clone https://github.com/annusiby5/python-scientific-calculator.git
```

Navigate to the project folder:

```bash
cd python-scientific-calculator
```

Run the application:

```bash
python scientific_calculator.py
```

## Usage

1. Launch the application.
2. Enter numbers and operators using the calculator buttons.
3. Click `=` to evaluate expressions.
4. Use scientific function buttons for advanced calculations.
5. Manage stored values using memory buttons.
6. View calculation history in the history panel.
7. Save history using the "Save History" button.
